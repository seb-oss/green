## Types

Specifying a type will cause the button to automatically render the appropriate variant for that type of button.

<Canvas>
  <Button type="button">Button Type</Button>
  <Button type="submit">Submit Type</Button>
  <Button type="reset">Reset Type</Button>
</Canvas>

## Variants

Variant can also be specified explicitly.

<Canvas>
  <Button variant="primary">Primary</Button>
  <Button variant="secondary">Secondary</Button>
  <Button variant="tertiary">Tertiary</Button>
  <Button variant="ghost">Ghost</Button>
</Canvas>

## Sizes

<Canvas>
  <Button>Default</Button>
  <Button size="small">Small</Button>
  <Button size="large">Large</Button>
</Canvas>

## Close button

When using the close button variant, take special care to provide an appropriate label!

<Canvas>
  <Button variant="close" aria-label="Close"></Button>
</Canvas>

## Disabled button

Buttons can be disabled by setting the `disabled` prop. In general, this should be strictly
avoided though. Rather than using disabled buttons, you should allow the user to click the
button and and instead provide an explanation of why the action is not available.

<Canvas>
  <Button variant="primary" disabled>
    Primary Button
  </Button>
  <Button variant="secondary" disabled>
    Secondary Button
  </Button>
  <Button variant="ghost" disabled>
    Ghost Button
  </Button>
</Canvas>
