## Types

Preferably use type to describe the function of a button. Use variant only when necessary.

<Canvas>
  <Button type="button">Button Type</Button>
  <Button type="submit">Submit Type</Button>
  <Button type="reset">Reset Type</Button>
</Canvas>

## Variants

Styling can also be achieved by class

<Canvas>
  <Button variant="primary">Primary</Button>
  <Button variant="secondary">Secondary</Button>
  <Button variant="ghost">Ghost</Button>
</Canvas>

## Close button

Close button with text for screen readers

<Canvas>
  <Button variant="close">
    <span className="sr-only">Close</span>
  </Button>
</Canvas>
