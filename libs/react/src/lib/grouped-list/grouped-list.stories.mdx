# Grouped List

[Source code](https://github.com/sebgroup/green/tree/main/libs/react/src/lib/grouped-list)
&nbsp;|&nbsp;
[Usage guidelines](https://designlibrary.sebgroup.com/components/lists)

## Overview

List are used to list many datapoints with labels in a structured way. They may be grouped under heading and they may have a link or button added to act on the data being displayed.

## Importing the component

In order to use the `GroupedList` component, you need to import the component:

```jsx
import { GroupedList } from '@sebgroup/green-react'

// Use as JSX element in your app
return (
  <GroupedList label="List heading">
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </GroupedList>
)
```

## Register transitional styles

This component does not need you to import styles from `Chlorophyll` it handles it with "transitional styles" that you need to register.

```jsx
import { registerTransitionalStyles } from '@sebgroup/green-react'
// Register the styles once in your apps entrypoint (main.tsx, app.tsx) file
registerTransitionalStyles()
```
