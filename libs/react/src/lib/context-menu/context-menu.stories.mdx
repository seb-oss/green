# Context Menu

This React component is a wrapper for the Context Menu component from Green Core. [You can read more about it here.](https://sebgroup.github.io/green/latest/core/?path=/docs/components-context-menu--page)

## Basic usage

In the example below, we are listening to the `onMenuItemClick` event on the `ContextMenu` component.

<Canvas>
<ContextMenu onMenuItemClick={console.log}>
  <MenuItem><strong>Item 1</strong></MenuItem>
  <MenuItem>Item 2</MenuItem>
  <MenuItem>Item 3</MenuItem>
</ContextMenu>
</Canvas>

A more typical usage would be to listen to `onClick` events on individual `MenuItem`s, while mapping to your specific data:

```jsx
<ContextMenu>
  {myListOfItems.map(item => (
    <MenuItem key={item.id} onClick={() => console.log(item.id)}>
      {item.name}
    </MenuItem>
  ))}
</ContextMenu>
```
