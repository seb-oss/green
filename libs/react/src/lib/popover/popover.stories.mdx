To use the component, import `Popover` like this:

```jsx

// Use as JSX element in your app
return (
  <Popover>
    <Button slot="trigger">Open popover</Button>
    <div style={{ padding: '1rem;' }}>Content in popover</div>
  </Popover>
)
```
