# Read this before using V-Angular

### V-Angular is being migrated to Green Design System and receive some updated but not be actively developed. Migrate to Green Core (or Green Angular and Green React)

V-Angular has been migrated to Green Design System in order to provide a more consistent and unified experience across all SEB products. The migration is still in progress and the components are being updated to match the new design system.
