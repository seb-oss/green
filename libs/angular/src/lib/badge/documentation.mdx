This component calls attention to a status/tag/category and highlights it from the surrounding content.

## Description

To display a status in an intuitive and easy way

## Usage

Use this component to display a status in a more prominent way than just typing out the status. Use the respective colour for alert messages.

## Importing module

```typescript
import { NgModule } from '@angular/core'
import { BrowserModule } from '@angular/platform-browser'
import { NggBadgeModule } from '@sebgroup/green-angular/src/lib/badge'

import { AppComponent } from './app.component'

@NgModule({
  declarations: [AppComponent],
  imports: [BrowserModule, NggBadgeModule],
  exports: [],
})
export class AppModule {}
```

## Badge Options

<Markdown>{`
| Input                 | Type        | Description                                                                                     |
| :-------------------- | :---------- | :---------------------------------------------------------------------------------------------- |
| badgeType             | \`BadgeType\` | The color of the component. Can be: \`info\`, \`success\`, \`warning\`, \`danger\`, \`light\`, and \`dark\` |
| isCloseable           | \`boolean\`   | Flag whether the component can be dismissed                                                     |
| closeText             | \`string\`    | Close text                                                                                      |
| customColor           | \`string\`    | Custom text color                                                                               |
| customBackgroundColor | \`string\`    | Custom background color                                                                         |
`}</Markdown>

### Outputs

<Markdown>{`
| Event       | Description                            |
| :---------- | :------------------------------------- |
| handleClose | Event emitted when badge is dismissed. |
`}</Markdown>

## Examples

To use this component, simply add an `ngg-badge` directive to an HTML `span` element:

```html
<span ngg-badge badgeType="success" class="mr-2">Badge</span>
<span ngg-badge badgeType="info" class="mr-2">Badge</span>
<span ngg-badge badgeType="warning" class="mr-2">Badge</span>
<span ngg-badge badgeType="danger" class="mr-2">Badge</span>
<span ngg-badge badgeType="light" class="mr-2">Badge</span>
<span ngg-badge badgeType="dark" class="mr-2">Badge</span>
```
