# Grouped list

[Source code](https://github.com/sebgroup/green/tree/main/libs/core/src/components/grouped-list)
&nbsp;|&nbsp;
[Usage guidelines](https://designlibrary.sebgroup.com/components/component-list)

The Grouped List component is used to list many datapoints with labels in a structured way. They may be grouped under heading and they may have a link or button added to act on the data being displayed.

## Use Webcomponent in Angular

This is a Webcomponent that comes from the `Green Core` library. Angular has support for using webcomponents directly in the template. To use this webcomponent in Angular, you need to do the following:

Add the `CUSTOM_ELEMENTS_SCHEMA` in the module that uses this component.

```ts
import { CUSTOM_ELEMENTS_SCHEMA } from '@angular/core'
import { NggCoreWrapperModule } from '@sebgroup/green-angular/shared/core-element/core-element.module'

@NgModule({
    // Add the NggCoreWrapperModule to the `imports` array
    imports: [NggCoreWrapperModule],
    // Add the CUSTOM_ELEMENTS_SCHEMA to the `schemas` array
    schemas: [CUSTOM_ELEMENTS_SCHEMA],
})
```

Import `GdsGroupedList`, `registerTransitionalStyles` and `NggCoreWrapperModule` from `@sebgroup/green-core` in the Angular component that uses it.

```ts
import { GdsGroupedList } from '@sebgroup/green-core'

// Transitional styles will make the component get 2016 design
import { registerTransitionalStyles } from '@sebgroup/green-core/transitional-styles'
// Call this fuction to register the transitional styles
registerTransitionalStyles()
```

Use the webcomponent in your template with the `*nggCoreElement` directive.

```html
<gds-grouped-list *nggCoreElement label="Grouped list label">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</gds-grouped-list>
```
