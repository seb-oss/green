# Progress Circle

A progress circle component that shows the progression of a system operation in a visual way.

## Importing Module

```typescript
import { NgModule } from '@angular/core'
import { BroswerModule } from '@angular/platform-browser'
import { NggProgressCircleModule } from '@sebgroup/green-angular/src/lib/progress-circle'

import { AppComponent } from './app.component'

@NgModule({
  declarations: [AppComponent],
  imports: [BroswerModule, NggProgressCircleModule],
  exports: [],
})
export class AppModule {}
```

## Progress Circle Options

<Markdown>{`
| Input | Type                     | Description                                         |
| :---- | :----------------------- | :-------------------------------------------------- |
| id    | \`string\`                 | _(Optional)_ ID of the progress circle              |
| theme | \`ProgressCircleThemes\` | Theme of the progress circle (default is \`warning\`) |
| value | \`number\`                 | Progress circle value in percentage                 |
`}</Markdown>

#### \*Progress Circle Themes

Available themes are: `success`, `warning`, `info`, `danger`, `light`, `dark`, and `disabled`. Choosing the disabled theme would disable the fill animation of the progress circle on component initialization.

## Example

The progress circle component is controlled with the `value` input. By default, the `value` represents the current percentage of progress, as the minimum and maximum values to be 0 and 100, respectively.

<Canvas>

Angular's content projection is used to insert whatever content is wanted to be displayed inside the progress circle component.

```html
<ngg-progress-circle>
  <div>content here</div>
</ngg-progress-circle>
```
