# Context Menu

A context menu.

## Description

Context menu with possibility to provide your template

## Importing module

If you're only using the NggContextMenuComponent component, you just need to import the `NggContextMenuModule`.

```typescript
import { NgModule } from '@angular/core'
import { BrowserModule } from '@angular/platform-browser'
import { NggContextModule } from '@sebgroup/green-angular'

import { AppComponent } from './app.component'

@NgModule({
  declarations: [AppComponent],
  imports: [BrowserModule, NggContextModule],
  exports: [],
})
export class AppModule {}
```

## Context Menu Options

### Inputs

| Input (attribute) | Description                             | Default |
| :---------------- | :-------------------------------------- | :------ |
| menuItemTemplate  | Template to user for context menu item. | default |

MenuItems use DropdownItem interface
| Input (attribute) | Description | Default |
| :---------------- | :------------------------------------------------------------------- | :------ |
| label | Which key to use as value. | `value` |
| value | Which key to use as display value when selected and in options list. | `label` |

### Outputs

| Event                  | Description                                                            |
| :--------------------- | :--------------------------------------------------------------------- |
| contextMenuItemClicked | Event emitted with the slider value when slider value has been changed |
