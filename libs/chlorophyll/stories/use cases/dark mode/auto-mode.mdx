## Auto detect light or dark mode

Since most of our user interfaces currently don't support dark mode, we can't just switch color theme for Green based on preferred color scheme set by the user for the browser as it would result in mixed color themes being used.
However, if you know that a particular app or flow runs standalone you can override the default theme and use the preferred color scheme by adding the class `auto-mode`. The example below will switch between dark and light mode base on your browser preferences.
