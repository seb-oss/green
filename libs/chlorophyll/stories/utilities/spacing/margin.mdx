# Margin

This utility class can be applied to any element to add margin.

**Class:** `m{value}-{breakpoint}-{negative}{spacing}` <br />
**Values:** `null`,`t` (top),`b` (bottom),`s` (start),`e` (end),`auto`,`x` (left and right),`y` (top and bottom) <br />
**Spacing:** `0-8`<br />
**Negative:** `null`, `n`<br />
**Responsive:** true
