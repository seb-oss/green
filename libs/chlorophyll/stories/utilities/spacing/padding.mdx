# Padding

This utility class can be applied to any element to add margin.

**Class:** `p{value}-{breakpoint}-{spacing}` <br />
**Values:** `null`,`t` (top),`b` (bottom),`s` (start),`e` (end),`x` (left and right),`y` (top and bottom) <br />
**Spacing:** `0-8`<br />
**Responsive:** true
