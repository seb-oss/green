# Flexbox align self

This utility class can be applied to child elements to elements with `display: flex` or `display: inline-flex`. Some common cases include `.col` etc.

**Class:** `align-self-{breakpoint}-{value}` <br />
**Values:** `auto`,`start`,`end`,`center`,`baseline`,`stretch` <br />
**Responsive:** true
