# Icons (experimental)

Use `sg-icon` which is the base class for icons together with a modifier eg. `sg-icon-calendar` to get a calendar icon.
Text content placed inside icons is hidden but will still be picked up by screen readers.

<Canvas>
  <i class="sg-icon sg-icon-calendar text-primary">
    Select date
  </i>
</Canvas>

## Icon color

Icons use css variable `--color` by default, so the color will shift depending on where and how it's being used.
For example if placed inside a button it will inherit the color from the button.

<Canvas>
  <div>
    <button type="button">
      <i class="sg-icon sg-icon-calendar">Select date</i>
    </button>
    <button type="button" class="primary">
      <i class="sg-icon sg-icon-calendar">Select date</i>
    </button>
    <button type="button" class="ghost">
      <i class="sg-icon sg-icon-calendar">Select date</i>
    </button>
  </div>
</Canvas>

### Set custom color

Use utility class for color i.e. `text-*` to set a custom color for icon.

<Canvas>
  <div>
    <i class="sg-icon sg-icon-calendar text-primary">Select date</i>
    <i class="sg-icon sg-icon-calendar text-secondary">Select date</i>
    <i class="sg-icon sg-icon-calendar text-light">Select date</i>
    <i class="sg-icon sg-icon-calendar text-dark">Select date</i>
    <i class="sg-icon sg-icon-calendar text-info">Select date</i>
    <i class="sg-icon sg-icon-calendar text-success">Select date</i>
    <i class="sg-icon sg-icon-calendar text-warning">Select date</i>
    <i class="sg-icon sg-icon-calendar text-danger">Select date</i>
  </div>
</Canvas>
