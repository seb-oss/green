# Group

Group inputs, buttons and other form elements together using <code>.group</code> class.

## Input and button

<Canvas>
  <div class="group">
    <input type="text" placeholder="Input field" />
    <button type="button">Normal button</button>
  </div>
</Canvas>

## Buttons aka button group

<Canvas>
  <div class="group">
    <button type="button">First button</button>
    <button type="button">Second button</button>
    <button type="button" class="active">
      Third button
    </button>
  </div>
</Canvas>
