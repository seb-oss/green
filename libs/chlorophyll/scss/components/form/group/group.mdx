# Group

Group inputs, buttons and other form elements together using <code>.group</code> class.

## Input and button


<div class="group">
  <input type="text" placeholder="Input field" />
  <button type="button">Normal button</button>
</div>

## Buttons aka button group

<div class="group">
  <button type="button">First button</button>
  <button type="button">Second button</button>
  <button type="button" class="active">Third button</button>
</div>

<div class="group size-sm">
  <button type="button" class="ghost">First button</button>
  <button type="button" class="ghost">Second button</button>
  <button type="button" class="ghost active">Third button</button>
  <button type="button" class="ghost">Forth button</button>
</div>
