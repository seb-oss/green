### Layout

Badges don't have any margin by default, instead they rely on the context in which they are used to provide the layout. Typically, you want multiple badges to have spacing (margin) in between, they should wrap if they don't fit on a single line. By placing badges inside an element with `.badge-container` we will get this behaviour by default.
