# Spinner

Spinners communicate to the user that their action is being processed, without telling them what is happening in the back end.

<Canvas of={SpinnerStories.Spinner} />

## Sizes

Sizes can changed with size classes

<span class="gds-spinner gds-spinner-sm" />
<span class="gds-spinner" />
<span class="gds-spinner gds-spinner-lg" />

## Layouts

Spinner can have cover or fullscreen layout

\*\*Note: To use cover class, the parent element should have position relative.

<div style={{ position: 'relative', height: '300px' }}>
  <div class="gds-spinner-wrapper gds-spinner-cover gds-spinner-backdrop">
    <span class="gds-spinner" />
  </div>
</div>

### Cover

Cover takes up the parent element size

```
class='gds-spinner-cover'
```

### Fullscreen

Fullscreen spinner takes up the entire screen

```
class='gds-spinner-fullscreen'
```

### Backdrop

```
class='gds-spinner-backdrop'
```
