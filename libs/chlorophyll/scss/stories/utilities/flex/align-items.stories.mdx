# Flexbox align self

This utility class can be applied to elements with `display: flex` or `display: inline-flex`. Some common cases include `.row` etc.

**Class:** `align-items-{breakpoint}-{value}` <br/>
**Values:** `start`,`end`,`center`,`baseline`,`stretch` <br/>
**Responsive:** true
