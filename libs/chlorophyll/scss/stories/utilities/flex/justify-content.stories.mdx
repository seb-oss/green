# Flexbox justify content

This utility class can be applied to elements with `display: flex` or `display: inline-flex`. Some common cases include `.row`, `.d-flex` etc.

**Class:** `justify-content-{breakpoint}-{value}` <br/>
**Values:** `start`,`between`,`center`,`evenly`,`around`,`end` <br/>
**Responsive:** true

## Examples

### Responsive
Distribute available space between flex items by default and justify items to end on medium screens and up.
<Canvas>
  <div class="d-flex justify-content-between justify-content-md-end">
    <div class="p-3 border border-info">Item 1</div>
    <div class="p-3 border border-info">Item 2</div>
    <div class="p-3 border border-info">Item 3</div>
  </div>
</Canvas>
