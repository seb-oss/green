# Navbar

Simple navbar component.

**CSS selector:** `<nav role="navigation">...</nav>` <br/>
**Responsive:** true

### Transparent navbar with heading and brand as link

<Canvas>
  <nav role="navigation">
    <a
      aria-label="Open seb.se in new tab"
      href="http://www.seb.se"
      target="_blank"
      class="brand"
    />
    <h1 class="mx-5">Page heading</h1>
  </nav>
</Canvas>
