# Group

Group inputs, buttons and other form elements together using <code>.group</code> class.

## Input and button

<Canvas>
  <div class="group">
    <input type="text" placeholder="Input field"/><button>Button</button>
  </div>
</Canvas>

## Buttons aka button group

<Canvas>
  <div class="group">
    <button>Button</button>
    <button>Button</button>
  </div>
</Canvas>

<Canvas>
  <div class="group size-sm">
    <button class="ghost">Button</button>
    <button class="ghost">Button</button>
    <button class="ghost">Button</button>
  </div>
</Canvas>
