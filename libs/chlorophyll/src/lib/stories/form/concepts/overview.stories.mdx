# Form layout and orientation

By default the flow in forms is vertical i.e. form elements such as inputs, buttons and labels will be stacked on top of each other and rendered in a vertical layout. They use full width by default and from medium-sized screens and up they fall back to `width: auto`.

**Selector:** `<form>` or `.form` <br/>
**Responsive:** true

## Disabled state

Styling is applied to `disabled`, `.disabled` and `aria-disabled="true"`. Note that `aria-disabled="true"` won't actually disable the control or element, it will only semantically indicate that it's disabled i.e. it will still be possible to select, click, focus and update it. Should be avoided but might be useful when action is required before proceeding without actually preventing user from trying.
