## Grouped list

The `<gds-grouped-list>` element accepts regular list items (`<li>`) as children.

<Canvas>
  <div style={{ height: '200px', dummy: registerTransitionalStyles() }}>
    <gds-grouped-list label="List label">
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
      <li>Item 4</li>
      <li>Item 5</li>
    </gds-grouped-list>
  </div>
</Canvas>

## Grouped list with values

By default, the list items are displayed as `flex` containers with `space-between` alignment.
