### The spacing scale is based on a 4px grid. 

The spacing scale is used for margin, padding, gap, radius and all the spacing values.

 ### *1 MAX  
 The MAX size is the maximum size of the spacing scale used when you want to create a circular shape.

 Example 
```
    <gds-card radius="max">MAX</gds-card>
```
