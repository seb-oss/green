## Design
Simple and recognizable design. Keep relations with form fields and buttons. The contained design version adds more visual punch and should be used then forms need more impact on the page. 

## Anatomy
The radio button have a group label (for all of the options), this may also have an additional label support text. Each option in the list have a label and optional label support text.
