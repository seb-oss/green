## Design
To keep the roundness throughout the design system the segmented control follows the design of the button. This component needs to take space, but visually and real-estate wise. Being a part of the navigational components it has a voice that speaks up. In combination with leading elements this component it speaks even higher.  

### Anatomy
This component consist of a container with item containers. Each item has a text label and an optional leading element.
