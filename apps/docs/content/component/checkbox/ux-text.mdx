## Labels
Write checkbox labels so customers can easily understand what checking or leaving the box blank means. This is especially important for stand-alone checkboxes, such as accepting terms or subscribing to emails with news and offers. If there is a header label, make sure it matches the individual checkbox label, whether grouped or stand-alone. Never use checkbox if the answer is yes or no.

## Stand-alone checkbox
A stand-alone checkbox should clearly indicate what happens when it’s checked or left blank. Use the “I” form. Avoid using negations as this forces users to check the box to prevent something from happening. A stand-alone checkbox should state a clear action or agreement, not ask a question.

## Multiple options
Ensure the label and options match. Provide clear instructions so users know they can select multiple options. Decide on a consistent style for option labels:
- Full sentences starting with a capital letter and ending with a full stop.
- A continuous sentence starting with lowercase, with only the last option ending in a full stop.
- Single words or short phrases with no full stop.
