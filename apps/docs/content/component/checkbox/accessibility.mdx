## Accessibility

To make it easier for users to use the checkbox, the checkbox label should also be clickable.
From accessibility perspective it is recommended to place checkboxes vertically to make it more obvious which label corresponds to which option.

## Review
Result from the latest accessibility review of the component (Chlorophyll, React): 2023-05-05

- Visual: Hover on selected fixed in latest version
