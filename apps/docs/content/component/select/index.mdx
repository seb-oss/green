## Select vs. Dropdown
- Select: For single-choice selections in forms, based on the native select element, with browser-determined appearance.
- Dropdown: For single and multiple-choice selections or actions outside forms, customizable with icons or badges and different layouts off the options. [View dropdown component = link]

## Anatomy
The Select (and other form controls) are divided in sections that are re-used between components. 
- Header: Label, Label support, contextual help, Extended support.
- Body: the actual field.
- Footer: error message.

## Behaviour
Styling for the list off options varies by browser, and long text in the select component is truncated with ellipses.
